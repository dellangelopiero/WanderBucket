# 🌍 WanderBucket

A simple web application that allows users to search for countries and create their personal **travel bucket list**.

The app fetches real country data using the **REST Countries API** and lets users add or remove destinations they want to visit.

---

## 🚀 Features

- Search countries by name
- Display country flags
- Add countries to a personal bucket list
- Remove countries from the list
- Clear the entire bucket list
- Data fetched from REST Countries API
- Built with pure JavaScript (no frameworks)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- REST Countries API

---
