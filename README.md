# 🌍 WanderBucket

A dynamic web application that allows users to search for countries and build a personalized travel bucket list.

The app integrates with the REST Countries API to fetch real-time data and provides an interactive interface for managing destinations.

---

## 🚀 Features

* 🔍 Real-time search filtering based on user input
* 🌐 Fetches live country data from an external API
* 🏳️ Displays country flags and names dynamically
* ➕ Add countries to a personal bucket list
* ❌ Remove individual countries from the list
* 🧹 Clear the entire bucket list instantly
* ⚡ Fully interactive UI built with vanilla JavaScript

---

## ✨ Key Highlights

* Designed and styled a clean, responsive user interface using CSS Flexbox
* Implemented dynamic DOM manipulation to render and update content
* Integrated a third-party REST API for real-time data fetching
* Built event-driven functionality for user interactions
* Managed application state directly in the browser
